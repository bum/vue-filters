# vue-filters

In [Vue 1 Filters](https://cli.vuejs.org/config/), there's a list of bultin filters those removed from Vue2.
If we want to reuse those filters, just add `vue-filters` into dependencies.

```
yarn add vue-filters
```

## Project setup
```
yarn install
```
